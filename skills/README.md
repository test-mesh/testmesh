# TestMesh Skills

Claude Code skills for writing, running, and debugging TestMesh integration test flows.

## Skills

| Skill | Triggers on |
|---|---|
| `testmesh-yaml-schema` | Writing/editing flow YAML, looking up action fields, assert syntax |
| `testmesh-action-patterns` | Writing flow steps — copy-pasteable patterns for every action type |
| `testmesh-debugging-flows` | Flow step failures, assertion errors, missing variables |
| `testmesh-mcp-tools` | Using `mcp__testmesh__*` tools to analyze, generate, and run flows |
| `testmesh-flow-layers` | Designing test suites — L1 contracts, L2 chains, L3 E2E, L4 edge cases |

## Install

```
/plugin install github:YOUR_ORG/testmesh
```

Or browse in Claude Code: `/plugin` → Discover → search "testmesh"
